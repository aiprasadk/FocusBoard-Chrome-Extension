## 🛠 HOW TO INSTALL (Chrome Browser)

Since this is a custom-built extension (not yet on the Web Store), follow these 4 simple steps to install it:

1.  **Download and Unzip:**
    * Download the `.zip` file I sent you.
    * Extract (unzip) the folder to a location on your computer (e.g., Documents or Desktop).

2.  **Open Chrome Extensions Page:**
    * Open Google Chrome.
    * Type `chrome://extensions/` in the address bar and press Enter.
    * Alternatively, go to Settings > Extensions.

3.  **Enable Developer Mode:**
    * In the top-right corner of the Extensions page, toggle the switch for **"Developer mode"** to ON.

4.  **Load the Extension:**
    * Click the **"Load unpacked"** button that appears in the top-left.
    * Select the unzipped folder containing the extension files (the folder that has `manifest.json` inside it).
    * Click "Select Folder" or "Open."
